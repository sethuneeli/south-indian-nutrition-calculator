# South Indian Nutrition Calculator

A comprehensive web-based nutrition calculator specifically designed for South Indian cuisine, featuring personalized meal planning for couples with individual calorie targets.

## üåü Features

### Core Functionality
- **Personalized Nutrition Tracking**: Individual and couple view with different calorie targets
- **South Indian Food Database**: Comprehensive database of traditional South Indian dishes
- **Real-time Nutrition Calculations**: Track calories, protein, carbs, and fat intake
- **Multiple User Profiles**: Support for individual users and couples (Sethu & Sangeetha)
- **Unit Toggle**: Switch between Metric (g/kg) and Imperial (lbs/oz) units

### Advanced Features
- **Beautiful PDF Generation**: 3-page meal planning PDFs with shopping lists and prep schedules
- **Personalized Diet Charts**: Based on ANUSHA KATTA's diet recommendations
- **Interactive Food Menu**: Collapsible menu with category filtering and search
- **Combined Food Logs**: View both partners' food entries in couple mode
- **Responsive Design**: Works on desktop, tablet, and mobile devices

### Couple-Specific Features
- **Individual Calorie Targets**: Sethu (1700 cal) & Sangeetha (1500 cal)
- **Combined Daily View**: Track both partners' nutrition simultaneously
- **Shared Meal Planning**: Coffee rituals and synchronized meal times
- **Weekly Prep Schedules**: Coordinated cooking and meal preparation

## üìñ Usage Guide

- All weights automatically convert in real-time
